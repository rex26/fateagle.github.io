使用hexo + github搭建博客(Windows)

第一步 安装Node.js

Node下载链接：
Node-v4.2.3 32 点我下载
Node-v4.2.3 64 点我下载
检查是否安装成功的命令：
node -v
npm -v

第二步 安装git

Git 下载链接：
git-2.6.3-64 点我下载
检查是否安装成功的命令：
git –version

第三步 github账号注册、新建库和添加SSH key

点我去注册就好了

新建库，命名为 github登录名.github.io
SSH key

Hexo部署：

  • npm install hexo-cli -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo server

ssh-keygen -t rsa -C “858790345@qq.com”

1.找到Git/usr/bin目录下的ssh-keygen.exe(如果找不到，可以在计算机全局搜索)
C:\Program Files\Git\usr\bin\ssh-keygen.exe

在C:\Users\Helios.ssh下，得到两个文件id_rsa和id_rsa.pub。

hexo generate
hexo deploy

hexo d后 ERROR Deployer not found: git

$ hexo d 
ERROR Deployer not found: git

npm install --save hexo-deployer-git
