react native 环境搭建(vs code)和工具使用

安装react native 智能提示

  1. npm install -g typings

  2. 先进入你的react-native项目中，在项目的根目录下:

    typings install dt~react –save

    typings install dt~react-native –save

npm安装指定的版本

npm install -g npm
npm install yarn@1.6.0 -g

ESlint找不到babel-eslint

到项目下面去安装,而不是全局安装babel-eslint

​ npm install babel-eslint –save-dev

​ npm install eslint-plugin-react –save-dev
